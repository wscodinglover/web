# 基本

  • * (opens new window) 选择所有元素
  • .class (opens new window) 选择具有给定类的所有元素
  • element (opens new window) 选择具有给定标签名称的所有元素
  • #id (opens new window) 选择具有给定 id 属性的单个元素
  • :hidden (opens new window) 选择所有隐藏的元素
  • :visible (opens new window) 选择所有可见的元素
  • :contains() (opens new window) 选择包含指定文本的所有元素
  • :empty (opens new window) 选择所有没有子元素的元素(包括文本节点)
  • :has() (opens new window) 选择至少包含一个与指定选择器匹配的元素的元素
  • :parent (opens new window) 选择具有至少一个子节点(元素或文本)的所有元素
  • parent > child (opens new window) 选择由父元素指定的子元素指定的所有直接子元素
  • ancestor descendant (opens new window) 选择作为给定祖先的后代的所有元素
  • prev + next (opens new window) 选择所有与 next 匹配且紧接在其前面的同级 prev 的下一个元素
  • prev ~ siblings (opens new window) 选择 prev 元素之后的所有同级元素，具有相同的父元素，并匹配过滤同级选择器

# 基本过滤器

  • :animated (opens new window) 在运行选择器时选择动画进度中的所有元素
  • :eq() (opens new window) 选择匹配集中索引 n 处的元素
  • :even (opens new window) 选择偶数元素，索引为零。 另见：奇数
  • :first (opens new window) 选择第一个匹配的 DOM 元素
  • :gt() (opens new window) 选择匹配集中索引大于索引的所有元素
  • :header (opens new window) 选择所有标题元素，如 h1、h2、h3 等
  • :lang() (opens new window) 选择指定语言的所有元素
  • :last (opens new window) 选择最后一个匹配的元素
  • :lt() (opens new window) 选择匹配集中索引小于索引的所有元素
  • :not() (opens new window) 选择与给定选择器不匹配的所有元素
  • :odd (opens new window) 选择奇数元素，索引为零。 另见：甚至
  • :root (opens new window) 选择作为文档根的元素
  • :target (opens new window) 选择由文档的 URI 的片段标识符指示的目标元素

# 属性

  • [name|="value"] (opens new window) 选择具有指定属性且值等于给定字符串或以该字符串开头后跟连字符 (-) 的元素
  • [name*="value"] (opens new window) 选择具有指定属性且值包含给定子字符串的元素
  • [name~="value"] (opens new window) 选择具有指定属性的元素，其值包含给定单词，由空格分隔
  • [name$="value"] (opens new window) 选择具有指定属性且值恰好以给定字符串结尾的元素。 比较区分大小写
  • [name="value"] (opens new window) 选择具有指定属性且值恰好等于特定值的元素
  • [name!="value"] (opens new window) 选择不具有指定属性或具有指定属性但不具有特定值的元素
  • [name^="value"] (opens new window) 选择具有指定属性且值恰好以给定字符串开头的元素
  • [name] (opens new window) 选择具有指定属性和任意值的元素
  • [name="value"] (opens new window) 匹配匹配所有指定属性过滤器的元素

# 子过滤器

  • :first-child (opens new window) 选择作为其父元素的第一个子元素的所有元素
  • :first-of-type (opens new window) 选择具有相同元素名称的兄弟姐妹中第一个的所有元素
  • :last-child (opens new window) 选择作为其父元素的最后一个子元素的所有元素
  • :last-of-type (opens new window) 选择具有相同元素名称的兄弟中最后一个的所有元素
  • :nth-child() (opens new window) 选择作为其父元素的第 n 个子元素的所有元素
  • :nth-last-child() (opens new window) 选择作为其父元素的第 n 个子元素的所有元素，从最后一个元素到第一个元素计数
  • :nth-last-of-type() (opens new window) 选择所有元素，这些元素是其父元素的第 n 个子元素，相对于具有相同元素名称的兄弟元素，从最后一个元素到第一个元素计数
  • :nth-of-type() (opens new window) 选择所有元素，这些元素是其父元素相对于具有相同元素名称的兄弟元素的第 n 个子元素
  • :only-child (opens new window) 选择作为其父元素的唯一子元素的所有元素
  • :only-of-type() (opens new window) 选择所有没有同名元素的元素

# Forms

  • :button (opens new window) 选择所有按钮元素和按钮类型的元素
  • :checkbox (opens new window) 选择复选框类型的所有元素
  • :checked (opens new window) 匹配所有选中或选择的元素
  • :disabled (opens new window) 选择所有禁用的元素
  • :enabled (opens new window) 选择所有启用的元素
  • :focus (opens new window) 如果当前获得焦点，则选择元素
  • :file (opens new window) 选择类型文件的所有元素
  • :image (opens new window) 选择图像类型的所有元素
  • :input (opens new window) 选择所有输入、文本区域、选择和按钮元素
  • :password (opens new window) 选择密码类型的所有元素
  • :radio (opens new window) 选择无线电类型的所有元素
  • :reset (opens new window) 选择重置类型的所有元素
  • :selected (opens new window) 选择所有选中的元素
  • :submit (opens new window) 选择提交类型的所有元素
  • :text (opens new window) 选择文本类型的所有输入元素

# Attributes

  • .attr() (opens new window) 获取匹配元素集中第一个元素的属性值
  • .prop() (opens new window) 获取匹配元素集中第一个元素的属性值
  • .removeAttr() (opens new window) 从匹配元素集中的每个元素中删除一个属性
  • .removeProp() (opens new window) 删除匹配元素集的属性
  • .val() (opens new window) 获取匹配元素集中第一个元素的当前值

# Data

  • jQuery.data() (opens new window) 存储与指定元素关联的任意数据。 返回设置的值
  • .data() (opens new window) 存储与匹配元素关联的任意数据
  • jQuery.hasData() (opens new window) 确定一个元素是否有任何与之关联的 jQuery 数据
  • jQuery.removeData() (opens new window) 删除以前存储的数据
  • .removeData() (opens new window) 删除以前存储的数据

# CSS

  • .addClass() (opens new window) 将指定的类添加到匹配元素集中的每个元素
  • .hasClass() (opens new window) 确定是否有任何匹配的元素被分配给给定的类
  • .removeClass() (opens new window) 从匹配元素集中的每个元素中删除单个类、多个类或所有类
  • .toggleClass() (opens new window) 根据类的存在或状态参数的值，从匹配元素集中的每个元素添加或删除一个或多个类
  • .css() (opens new window) 获取匹配元素集中第一个元素的计算样式属性
  • jQuery.cssHooks (opens new window) 直接连接到 jQuery 以覆盖特定 CSS 属性的检索或设置方式、标准化 CSS 属性命名或创建自定义属性
  • jQuery.cssNumber (opens new window) 包含所有可以在没有单位的情况下使用的 CSS 属性的对象。.css() 方法使用此对象来查看它是否可以将 px 附加到无单位值
  • jQuery.escapeSelector() (opens new window) 转义任何在 CSS 选择器中具有特殊含义的字符

# Dimensions

  • .height() (opens new window) 获取匹配元素集中第一个元素的当前计算高度
  • .innerHeight() (opens new window) 获取匹配元素集中第一个元素的当前计算高度，包括填充但不包括边框
  • .innerWidth() (opens new window) 获取匹配元素集中第一个元素的当前计算内部宽度，包括填充但不包括边框
  • .outerHeight() (opens new window) 获取匹配元素集中第一个元素的当前计算外部高度(包括填充、边框和可选的边距)
  • .outerWidth() (opens new window) 获取匹配元素集中第一个元素的当前计算外部宽度(包括填充、边框和可选的边距)
  • .width() (opens new window) 获取匹配元素集中第一个元素的当前计算宽度

# Offset

  • .offset() (opens new window) 获取匹配元素集中第一个元素相对于文档的当前坐标
  • .offsetParent() (opens new window) 获取定位的最近的祖先元素
  • .position() (opens new window) 获取匹配元素集中第一个元素相对于偏移父元素的当前坐标
  • .scrollLeft() (opens new window) 获取匹配元素集中第一个元素的滚动条的当前水平位置
  • .scrollTop() (opens new window) 获取匹配元素集中第一个元素的滚动条的当前垂直位置，或为每个匹配元素设置滚动条的垂直位置

# DOM 插入，环绕

  • .wrap() (opens new window) 在匹配元素集中的每个元素周围包装一个 HTML 结构
  • .wrapAll() (opens new window) 在匹配元素集中的所有元素周围包装一个 HTML 结构
  • .wrapInner() (opens new window) 围绕匹配元素集中每个元素的内容包装一个 HTML 结构

# DOM 插入，内部

  • .append() (opens new window) 在匹配的元素集中，插入由参数指定的内容插入每个元素的末尾
  • .appendTo() (opens new window) 将匹配元素集中的每个元素插入目标末端
  • .html() (opens new window) 在匹配元素集中获取第一个元素的HTML内容
  • .prepend() (opens new window) 插入由参数指定的内容插入匹配元素集中每个元素的开始
  • .prependTo() (opens new window) 将匹配元素集中的每个元素插入目标开始
  • .text() (opens new window) 在匹配的元素集（包括其后代）中获取每个元素的组合文本内容

# DOM 插入，外部

  • .after() (opens new window) 在匹配元素集中的每个元素之后，插入由参数指定的内容
  • .before() (opens new window) 在匹配元素集中的每个元素之前，插入由参数指定的内容
  • .insertAfter() (opens new window) 在目标之后插入匹配元素集中的每个元素
  • .insertBefore() (opens new window) 在目标之前插入匹配元素集中的每个元素

# DOM 移除

  • .detach() (opens new window) 从DOM上删除一组匹配的元素
  • .empty() (opens new window) 从DOM中删除一组匹配元素的所有子节点
  • .remove() (opens new window) 从DOM上删除一组匹配的元素
  • .unwrap() (opens new window) 从DOM上删除一组匹配元素的父母，将匹配的元素留在其位置

# DOM 替换

  • .replaceAll() (opens new window) 用匹配的元素集替换每个目标元素
  • .replaceWith() (opens new window) 用提供的新内容替换一组匹配元素中的每个元素，然后返回已删除的元素集

# jQuery 遍历

# 过滤

  • .eq() (opens new window) 将匹配的元素集减少到指定索引处的集合
  • .filter() (opens new window) 将匹配的元素集减少到匹配选择器或传递函数测试的元素
  • .first() (opens new window) 将匹配的元素集减少到集合中的第一个
  • .has() (opens new window) 将匹配的元素集减少到具有与选择器或DOM元素相匹配的后代的元素
  • .is() (opens new window) 检查当前匹配的元素与选择器，元素或jQuery对象的元素集，如果至少其中一个元素与给定参数匹配，则返回true
  • .last() (opens new window) 将匹配的元素集减少到集合中的最后一个元素
  • .map() (opens new window) 将当前匹配的每个元素通过函数传递，从而产生包含返回值的新jQuery对象
  • .not() (opens new window) 从匹配元素集中删除元素
  • .slice() (opens new window) 将匹配的元素集减少为由一系列索引指定的子集

# 杂项遍历

  • .add() (opens new window) 创建一个新的jQuery对象，其中添加到匹配元素集中的元素
  • .addBack() (opens new window) 将堆栈上的上一个元素添加到当前集合中，并由选择器进行过滤
  • .andSelf() (opens new window) 将堆栈上的先前元素添加到当前集合中
  • .contents() (opens new window) 在一组匹配元素中获取每个元素的孩子，包括文本和评论节点
  • .each() (opens new window) 在jQuery对象上迭代，为每个匹配的元素执行一个函数
  • .end() (opens new window) 结束当前链中最新的过滤操作，并将匹配元素集返回其先前状态

# 树遍历

  • .children() (opens new window) 在一组匹配元素中获取每个元素的孩子，并被选择器进行过滤
  • .closest() (opens new window) 对于集合中的每个元素，通过测试元素本身并通过其祖先在DOM树中的祖先进行遍历来获取与选择器匹配的第一个元素
  • .find() (opens new window) 在当前匹配元素集中获取每个元素的后代，并由选择器，jQuery对象或元素过滤
  • .next() (opens new window) 在匹配的元素集中，立即获取每个元素的兄弟姐妹。 如果提供了选择器，则仅在匹配该选择器的情况下才能检索下一个兄弟姐妹
  • .nextAll() (opens new window) 在一组匹配元素中获取每个元素的所有以下兄弟姐妹，并由选择器进行过滤
  • .nextUntil() (opens new window) 将每个元素的所有以下兄弟姐妹添加到，但不包括通过选择器，DOM节点或jQuery对象匹配的元素

  • callbacks.has() (opens new window) 确定列表是否附加了任何回调。如果回调作为参数提供，请确定它是否在列表中
  • callbacks.lock() (opens new window) 将回调列表锁定在其当前状态
  • callbacks.locked() (opens new window) 判断回调列表是否被锁定
  • callbacks.remove() (opens new window) 从回调列表中删除回调或回调集合

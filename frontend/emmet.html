    # Emmet 备忘清单

    Emmet (opens new window) 是一个用于提升 HTML 和 CSS 代码编写的 Web 开发人员工具包，它允许您使用著名的 CSS 选择器以光速编写大型 HTML 代码块。

    # Emmet 语法

    # 入门

    让我们开始将您的开发提高到光速。

    • Visual Studio 代码中的 Emmet (opens new window) (code.visualstudio.com)
    • Sublime Text 的 Emmet 2 (opens new window) (github.com)
    • Emmet for Coda (opens new window) (emmet.io)
    • Emmet for Atom (opens new window) (github.com)
